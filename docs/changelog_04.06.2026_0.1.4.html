🏷 Версия 0.1.4 — что нового с 0.1.0

Линейка 0.1.x посвящена выходу приложения «в продакшн»: запуск на сервере, обслуживание и удобство эксплуатации.

  • Готовое развёртывание на Windows Server (waitress + скрипты).
  • Ручное обновление приложения с GitHub одним скриптом.
  • Резервное копирование базы данных.
  • Отображение номера версии в интерфейсе.

Этапы разработки (0.1.1 → 0.1.4)

0.1.1Развёртывание на Windows Server

  • ДОБАВЛЕНО serve.py — производственный запуск через WSGI-сервер waitress (адрес 0.0.0.0, порт настраивается переменными HOST/PORT/THREADS).
  • waitress добавлен в requirements.txt.
  • Папка scripts/ со скриптами: setup.ps1 (окружение, зависимости, открытие порта в брандмауэре), start.ps1 (запуск сервера), update.ps1 (обновление с GitHub), install-service.ps1 / uninstall-service.ps1 (служба Windows через NSSM), а также .bat-лаунчеры для запуска двойным кликом.
  • scripts/README.md — пошаговая инструкция по развёртыванию и обновлению.
  • .gitignore: исключены logs/ и backups/; в README добавлен раздел «Развёртывание на сервере».
serve.py · scripts/* · requirements.txt · README.md · .gitignore

0.1.2Резервное копирование и кодировка скриптов

  • ДОБАВЛЕНО scripts/backup.ps1 (+ backup.bat) — ручная резервная копия base.db в папку backups\ с отметкой даты/времени; опция -Keep N для хранения только N последних копий.
  • ИСПРАВЛЕНО Все скрипты .ps1 пересохранены в кодировке UTF-8 с BOM. Без BOM встроенный в Windows Server Windows PowerShell 5.1 читает кириллицу как ANSI, и скрипты не запускаются.
  • Корректность проверена под реальным powershell.exe 5.1: все скрипты парсятся без ошибок, backup.ps1 отрабатывает.
scripts/backup.ps1 · scripts/backup.bat · scripts/*.ps1
