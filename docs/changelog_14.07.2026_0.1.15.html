О документе

Продолжение истории после версии 0.1.14. Версия 0.1.15 закрывает замечание пользователей по бумажной печати: отмеченные чекбоксы в заполненных формах были слишком бледными и могли плохо читаться после распечатки.

Версия 0.1.15 — контрастные галочки в печатных формах

Печатные версии обеих форм теперь используют собственное CSS-отображение чекбоксов: чёткий квадрат и более жирная чёрная отметка вместо нативной браузерной галочки, которая на части принтеров печаталась серой.

  • Экранное отображение форм не изменено.
  • Изменение применяется только в режиме печати через @media print.
  • Исправление распространяется на обе формы, включая динамически добавляемые блоки.

Этапы разработки (0.1.15)

0.1.15Улучшение читаемости печатных чекбоксов

  • ИСПРАВЛЕНО В печатном CSS заменено нативное отображение чекбоксов на управляемое приложение CSS-оформление.
  • Для печати задан чёрный контур чекбокса, фиксированный размер и жирная чёрная галочка через псевдоэлемент ::after.
  • Сохранены параметры print-color-adjust: exact и -webkit-print-color-adjust: exact, чтобы браузер не осветлял элементы при выводе на бумагу.
  • ДОКУМЕНТАЦИЯ Создан changelog выпуска 0.1.15; версия приложения поднята до 0.1.15 и отображается в интерфейсе.
